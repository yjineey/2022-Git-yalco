`yalco 인강 정리`
* git commit -am '커밋내용'

* 로컬에서 커밋 reset 위치로 가고 싶을 때, 내가 선택한거 전으로 이동
git reset --hard (커밋)

* 원격저장소에서 현재 reset 위치로 가고 싶을 때
git push -f origin main

* merge 와 rebase 차이
merge는 이전 branch history가 유지된다 (main으로 이동)
rebase는 merge시 이전 branch history가 삭제 된다 (rebase 브랜치로 이동)
